# Contributing

Thanks for contributing to AgentWatch.

## Development

```bash
npm install
npm start
```

## Build

```bash
npm run build:win
```

## Pull Requests

- Keep changes focused.
- Include steps to test.
- Add screenshots/GIFs for UI changes.
- Do not include real auth cookies/tokens in logs, screenshots, or commits.

## Issue Reports

Please include:

- Windows version
- Node/npm version
- App version
- Reproduction steps
- Sanitized debug logs when relevant
